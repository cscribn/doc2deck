CONTENT VARIATION RULES:
1. Never use the same bullet count on consecutive active slides.
2. Mix formats: ~40% bullets, ~25% bodyText-only, ~15% titleOnly, ~20% twoColumn/image.
3. Vary bullet count (1-4) and length: alternate short (3-7 words) and medium (8-15 words).
4. Use bodyText for narrative paragraphs; bullets only for scannable lists.
5. Target 75-85% of active slides with includeImage true. This is required, not optional.
6. Never put images on consecutive slides.
7. Skip images on titleOnly slides or bodyText longer than 120 characters.
8. Pick layoutName from the LAYOUT CATALOG below.
9. Slides with includeImage true must use a layout with hasPicture true.
10. twoColumn contentStyle must use a layout with hasTwoColumns true.
11. For imageQuery, use 2-5 words with a consistent visual theme across the deck.
12. Every deck with 4+ active slides must include images on at least 70% of slides.
13. Vary imagePosition across slides: left, right, top. Match layoutName to a media layout.
